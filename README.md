# nimbusflow-landing-page

nimbusflow is a modern, responsive SaaS landing page designed to showcase an AI-powered analytics and workflow automation platform.

## Features
- Responsive layout for mobile, tablet, and desktop
- Modern UI inspired by SaaS design patterns
- Built using Tailwind CSS utility-first approach
- Clean component-based structure
- Gradient backgrounds and card-based sections
- Optimized typography and spacing

## Tech Stack
- HTML5
- Tailwind CSS

## Responsiveness
The landing page is fully responsive and adapts seamlessly across different screen sizes using Tailwind CSS responsive utilities.
